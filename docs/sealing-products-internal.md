# Sealing — Internal Product Reference

**INTERNAL ONLY. Not for the public website.** Public copy stays generic ("water-based sealant," "premium solvent-based sealant," "non-slip additive," "polymeric jointing sand"). No brand names, no mix ratios on the live site.

## Sealants
- **Rainy season (water-based):** ClariShield Water-Based Clear Concrete Sealer
- **Dry / winter (solvent-based):** Cobble Coat (Proline / Sherwin-Williams)

## Non-slip additive
- **SharkGrip Slip-Resistant Additive** — patios ONLY (travertine, marble, smooth surfaces). Never on driveways.

## Cleaning
- Cleaning solution: gentle on grass/plants (mix ratio internal)

## Jointing
- Polymeric sand

## Process notes
